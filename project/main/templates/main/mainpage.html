템플릿(template) 언어

    파이썬의 변수, 문법을 HTML에서 사용할 수 있도록 제공하는 직관적인 언어

  • 변수➡{{변수명}}
  • 변수의 속성➡{{변수명.속성명}}
  • list의 각 인덱스➡{{list.index}}
  • dict의 key값➡{{dict.key}}
  • 조건문 형태 {% if %} {% else %} {% endif %}
  • 반복문 형태 {% for num in nums %} {{ num }} {% endfor %}
  • 템플릿 필터 {{ 변수명|필터 }}
  • 템플릿 주석 {# 주석 #}
템플릿 상속, nav bar 분리

    반복되는 코드 상속하기

  • project 폴더 내, templates 폴더 + base.html생성
  • {%block content%} : 재사용 가능한 블록 범위를 정의
  • {%endblock%} : 블록의 끝
  • {%extends 'base.html%} : 업데이트
  • 장고가 base.html의 경로를 찾을 수 있게 설정
